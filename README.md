\# Spring Boot REST API Learning Project



\## Purpose

This project is for learning core backend concepts using Java and Spring Boot.



\## What This Project Covers



\### Spring Boot

\- Application bootstrapping

\- Dependency injection

\- REST controllers

\- Configuration and project structure

\- Running an embedded server



\### REST API

\- Creating REST endpoints

\- Handling HTTP methods (GET, POST, PUT, DELETE)

\- Request parameters and path variables

\- Response bodies and HTTP status codes

\- Basic API design principles



\### JUnit Testing

\- Unit testing with JUnit 5

\- Testing Spring Boot controllers

\- Using MockMvc to simulate HTTP requests

\- Verifying responses and status codes

\- Ensuring endpoints behave correctly



\## Tech Stack

\- Java

\- Spring Boot

\- Spring Web

\- JUnit 5

\- MockMvc

\- Gradle



\## Goal

Build a solid understanding of how Spring Boot applications are structured, how REST APIs are implemented, and how automated tests validate application behavior.
